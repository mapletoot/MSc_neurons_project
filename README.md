# Uncovering Patterns in Projection and Gene Expression of neurons in layer 5a of the Entorhinal cortex in mouse brains
Important instructions:
1. In order for the notebooks to run, upload 2 files called `master_detailed_commented.csv` and `ec_obj_imputed_log2.h5ad` into the Data directory.
2. The notebooks are named according to the section that they relate to in the pdf report.
3. Run the notebooks in numerical order so that objects and data created in previous notebooks can be generated. 

# Reproducing the Analysis
Note that to move through the notebooks, you will need to close the brain render images that can appear as a pop up in a different screen. Unfortunately, on closing these renders the render seems to freeze. This doesn't prevent new renders appearing, but it does mean that the screen showing that render becomes frozen.
